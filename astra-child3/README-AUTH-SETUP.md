## 🔐 تنظیمات Google OAuth

### مرحله 1: دسترسی به پنل تنظیمات

1. وارد پنل مدیریت وردپرس شوید
2. از منوی سمت راست، روی **تنظیمات احراز هویت** کلیک کنید
3. تب **Google OAuth** را انتخاب کنید

### مرحله 2: ایجاد پروژه در Google Cloud Console

1. به آدرس [console.cloud.google.com](https://console.cloud.google.com) بروید
2. روی "Select a project" کلیک کنید و "New Project" را انتخاب کنید
3. نام پروژه را وارد کنید (مثلاً "My Website Auth")
4. روی "Create" کلیک کنید

### مرحله 3: فعال‌سازی Google+ API

1. از منوی سمت چپ، به **APIs & Services > Library** بروید
2. "Google+ API" را جستجو کنید
3. روی آن کلیک کنید و "Enable" را بزنید

### مرحله 4: تنظیم OAuth Consent Screen

1. از منوی سمت چپ، به **APIs & Services > OAuth consent screen** بروید
2. User Type را "External" انتخاب کنید
3. اطلاعات برنامه را وارد کنید:
   - App name: نام سایت شما
   - User support email: ایمیل پشتیبانی
   - Developer contact information: ایمیل توسعه‌دهنده
4. روی "Save and Continue" کلیک کنید

### مرحله 5: ایجاد OAuth Client ID

1. از منوی سمت چپ، به **APIs & Services > Credentials** بروید
2. روی "Create Credentials" کلیک کنید
3. "OAuth client ID" را انتخاب کنید
4. Application type را "Web application" انتخاب کنید
5. نام را وارد کنید (مثلاً "Website Login")
6. در قسمت **Authorized JavaScript origins** آدرس سایت خود را اضافه کنید:
   \`\`\`
   https://yourdomain.com
   \`\`\`
7. در قسمت **Authorized redirect URIs** این آدرس را اضافه کنید:
   \`\`\`
   https://yourdomain.com/google-callback/
   \`\`\`
8. روی "Create" کلیک کنید

### مرحله 6: کپی اطلاعات به وردپرس

1. Client ID و Client Secret را کپی کنید
2. به پنل **تنظیمات احراز هویت** در وردپرس برگردید
3. Client ID و Client Secret را در فیلدهای مربوطه وارد کنید
4. روی "ذخیره تنظیمات" کلیک کنید

### تغییر اطلاعات Google OAuth

برای تغییر Client ID یا Client Secret:

1. به پنل **تنظیمات احراز هویت** بروید
2. فیلدهای Client ID و Client Secret را ویرایش کنید
3. روی "ذخیره تنظیمات" کلیک کنید

## 📱 تنظیمات فراز اس‌ام‌اس

### مرحله 1: ثبت‌نام در فراز اس‌ام‌اس

1. به آدرس [farazsms.com](https://farazsms.com) بروید
2. روی "ثبت‌نام" کلیک کنید
3. فرم ثبت‌نام را تکمیل کنید

### مرحله 2: دریافت API Key

1. وارد پنل کاربری فراز اس‌ام‌اس شوید
2. از منو، به **تنظیمات > API** بروید
3. API Key خود را کپی کنید
4. اگر API Key ندارید، روی "ایجاد API Key" کلیک کنید

### مرحله 3: دریافت شماره خط

1. از منوی پنل، به **خطوط من** بروید
2. شماره خط ارسال پیامک را کپی کنید
3. اگر خط ندارید، از قسمت "خرید خط" اقدام کنید

### مرحله 4: تنظیم در وردپرس

1. به پنل **تنظیمات احراز هویت** در وردپرس بروید
2. تب **فراز اس‌ام‌اس** را انتخاب کنید
3. گزینه "فعال‌سازی ورود با موبایل" را فعال کنید
4. API Key را در فیلد مربوطه وارد کنید
5. شماره خط را در فیلد "شماره ارسال‌کننده" وارد کنید
6. روی "ذخیره تنظیمات" کلیک کنید

### تنظیمات اضافی

- **الزامی بودن شماره موبایل**: اگر می‌خواهید شماره موبایل برای ثبت‌نام الزامی باشد، این گزینه را فعال کنید

### نکات مهم

- حساب شما باید دارای اعتبار کافی برای ارسال پیامک باشد
- شماره خط باید تایید شده و فعال باشد
- برای تست، می‌توانید از پنل فراز اس‌ام‌اس یک پیامک آزمایشی ارسال کنید

## 👤 نمایش تصویر پروفایل در هدر

### عملکرد خودکار

سیستم به صورت خودکار تصویر پروفایل کاربران را در هدر سایت نمایش می‌دهد:

#### کاربران وارد شده:
- اگر کاربر با Google وارد شده باشد، تصویر پروفایل Google نمایش داده می‌شود
- اگر تصویر Google موجود نباشد، آواتار وردپرس نمایش داده می‌شود
- تصویر به صورت دایره‌ای با حاشیه سفید نمایش داده می‌شود

#### کاربران وارد نشده:
- یک آیکون پیش‌فرض کاربر نمایش داده می‌شود

### تغییر تصویر پروفایل

کاربران می‌توانند تصویر پروفایل خود را از دو روش تغییر دهند:

#### روش 1: از طریق Google
1. به حساب Google خود بروید
2. تصویر پروفایل را تغییر دهید
3. دوباره با Google به سایت وارد شوید
4. تصویر جدید به صورت خودکار نمایش داده می‌شود

#### روش 2: از طریق Gravatar
1. به [gravatar.com](https://gravatar.com) بروید
2. با ایمیل خود ثبت‌نام کنید
3. تصویر پروفایل را آپلود کنید
4. تصویر در تمام سایت‌های وردپرسی که از آن ایمیل استفاده می‌کنید، نمایش داده می‌شود

## 🔧 عیب‌یابی

### مشکل: تصویر پروفایل نمایش داده نمی‌شود

**راه‌حل‌ها:**
1. از کش مرورگر پاک کنید (Ctrl+Shift+R)
2. مطمئن شوید کاربر وارد سیستم شده است
3. بررسی کنید که تصویر پروفایل در Google یا Gravatar تنظیم شده باشد

### مشکل: ورود با Google کار نمی‌کند

**راه‌حل‌ها:**
1. Client ID و Client Secret را بررسی کنید
2. مطمئن شوید Redirect URI در Google Cloud Console درست تنظیم شده است
3. Google+ API را فعال کنید

### مشکل: پیامک ارسال نمی‌شود

**راه‌حل‌ها:**
1. API Key و شماره خط را بررسی کنید
2. اعتبار حساب فراز اس‌ام‌اس را چک کنید
3. مطمئن شوید شماره خط فعال است
